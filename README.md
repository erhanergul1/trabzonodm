# TrabzonODM Projesi

## Proje Açıklaması

TrabzonODM, Trabzon bölgesinde veri yönetimi ve analizine olanak tanıyan bir projedir. Bu proje, kullanıcıların bölgesel verileri etkili bir şekilde toplamasına, depolamasına ve analiz etmesine yardımcı olmak için tasarlanmıştır.

## Özellikler
- Kullanıcı dostu arayüz
- Hızlı ve etkili veri yönetimi
- Detaylı raporlama ve analiz araçları

## Başlangıç

### Gereksinimler
- Python 3.6 veya üzeri
- Flask

### Kurulum
1. Repositori klonlayın: `git clone https://github.com/erhanergul1/trabzonodm.git`
2. Gerekli kütüphaneleri yükleyin: `pip install -r requirements.txt`
3. Uygulamayı başlatın: `python app.py`

## Kullanım
Uygulama başlatıldıktan sonra tarayıcınızda `http://127.0.0.1:5000` adresine gidin.
